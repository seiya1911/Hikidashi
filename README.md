# Hikidashi

## サイト概要
もの管理アプリ

### サイトテーマ
持っているものを管理するためのアプリ

### テーマを選んだ理由
自分が服が好きでついたくさん買ってしまい、収納スペースを圧迫してしまうので、  
持っているものをカテゴリーごとでわかりやすく管理したいなと思ってテーマに選びました。

### ターゲットユーザ
- たくさん買い物をしてしまう人
- 持っているものをついついまた買ってしまう人
- 収納スペースが少ない人

### 主な利用シーン
お買い物をするときに、そのものが必要かどうか判断したいとき。  
以前に買ったものと買おうとしているものがかぶっていないか確認したいとき。  
ものを減らそうと考えているときに持っているものを俯瞰して見たいとき。
